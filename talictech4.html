Case Studies

Talic Tech being only established recently currently only covers Web Development and basic IT services. There for our projects include Web Development for Sonic Fitness, Yass Youth Association, Ygrene Energy and Yass Appliances. Sonic Fitness is a fitness gym establishment in the town of Yass NSW. Sonic Fitness was the first project for Talic Tech which was a learning process for lunching an actual live website. Sonic Fitness features a basic layout with much content that was displayed in a unique and creative way that meat functionality specification that the owner of Sonic Fitness required. Sonic Fitness features an embedded linked site to a signup application so that users can purchase gym memberships online. Ygrene Energy is an Australian Solar company based in Yass the provides services all over Australia. The Ygrene Energy website was where more advanced web development methods where introduced. The Ygrene Energy website encompasses modern and contemporary design features that keep up with modern trends and for the first time has been completely re-constructed to be viewable for mobile devices and screens with different resolutions.
